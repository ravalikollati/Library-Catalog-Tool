## Project Name

Library Catalog Tool

## Project Overview

The Library Catalog Tool is a web-based application designed to manage library book records efficiently. It enables users to add books, display book information, monitor book availability, and delete records through an easy-to-use interface.

## Project Scope

The application simplifies library management by providing a digital catalog where books can be organized and maintained without the need for a database.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

### 1. Add Book

Users can add a new book by entering:

* Book Name
* Author Name
* Book Status (Available/Issued)

### 2. Book Catalog

Displays all books in a table with:

* Book Name
* Author
* Availability Status

### 3. Book Status

Shows whether a book is currently:

* Available
* Issued

### 4. Delete Book

Allows users to remove a selected book from the catalog using the Delete button.

### 5. Dynamic Updates

The book list updates automatically whenever books are added or deleted.

## Working Procedure

1. Open the application using index.html.
2. Enter the book details.
3. Click the **Add Book** button.
4. The book appears in the catalog table.
5. View the availability status of each book.
6. Delete books when required using the Delete button.

## Output Description

The output screen displays:

* Library Catalog Dashboard
* Book Entry Form
* Book List Table
* Author Information
* Book Availability Status
* Delete Button for each record
