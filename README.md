## Project Name

Library Catalog Tool

## Project Scope

The Library Catalog Tool is a web-based application developed using HTML, CSS, and JavaScript. It allows users to manage a library's book collection by adding new books, viewing available and issued books, and deleting book records through a simple and interactive interface.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Add New Books
* Display Book Details
* View Book Author
* Track Book Status (Available/Issued)
* Delete Book Records
* Responsive User Interface
